# Cumulus Platform Documentation

Welcome to the documentation for the **Cumulus Agentic Platform** — a fully
managed, multi-tenant delivery platform for agents, agentic workflows, and
agentic applications.

This site provides public-facing product documentation to help users and
administrators get the most out of the platform.

## What you'll find here

- **[User Guide](user-guide/getting-started.md)** — Start here to learn the
  basics and core concepts.
- **[Knowledge Base](knowledge-base/index.md)** — How-to articles on common
  administrative tasks.
- **[API Reference](api-reference.md)** — Overview of how to integrate with the
  platform programmatically.
- **[FAQ](faq.md)** — Quick answers to common questions.
- **[Incident Reports](incidents/index.md)** — Our transparency log for service
  incidents.
