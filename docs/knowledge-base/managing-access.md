# Managing Access

Access to the platform is provisioned and managed by the Naviant team. Access
is not self-service today — changes are made on your behalf by request.

## Roles

The platform uses role-based access. Common roles include:

- **Member** — Uses workflows and agents within their team.
- **Elevated** — Additional permissions for designated users.

The exact roles available to your organization are configured by the Naviant
team.

## Requesting access

To add a new user, submit a request through
[Naviant Customer Support](https://naviant.com/customer-support/) with the
person's details and the role they need. The Naviant team provisions the
account and confirms when it's ready.

## Removing access

When someone no longer needs access, submit a request through
[Naviant Customer Support](https://naviant.com/customer-support/) to revoke it.
Access changes are applied promptly.

!!! note
All access changes are handled by the Naviant team. Submit requests through
[Naviant Customer Support](https://naviant.com/customer-support/).
