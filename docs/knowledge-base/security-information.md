# Security Information

This page provides a public overview of the Cumulus Platform security
practices.

## Security practices

- Tenant-level data isolation is enforced so each organization operates in its
  own logical boundary.
- Data is encrypted in transit.
- Access to customer environments follows least-privilege principles.
- Administrative and operational access is logged and auditable.
- Security controls are aligned to SOC 2 principles for security,
  availability, and confidentiality.

## Compliance posture

The platform is built to SOC 2 controls. We anticipate **SOC 2 Type 1**
certification by the end of Q2 and **SOC 2 Type 2** by the end of Q3.
Additional certifications such as HIPAA and ISO 42001 are part of the roadmap.
