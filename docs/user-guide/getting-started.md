# Getting Started

This guide introduces the Cumulus Platform and helps new users get oriented.

## Overview

Cumulus is a managed delivery platform for agents, agentic workflows, and
agentic applications — letting your organization run them without managing the
underlying infrastructure. Today the platform is delivered as an **API-first
service**, so you integrate Cumulus directly into your own systems and
orchestration layers.

!!! info "Looking ahead"
Cumulus is under active development along an ambitious product roadmap. A
dedicated web console is on the way, bringing self-service management,
real-time metrics, and deeper operational transparency on top of the same
API you build against today.

## Getting connected

1. Request access through
   [Naviant Customer Support](https://naviant.com/customer-support/).
2. The Naviant team provisions your tenant and provides your API base URL and
   credentials.
3. Authenticate to the API and make your first request — see the
   [API Reference](../api-reference.md).

!!! note
Access is provisioned by the Naviant team. If you need help getting set up,
contact
[Naviant Customer Support](https://naviant.com/customer-support/).

## Your first integration

Once your credentials are active you can:

- Discover the workflows and agents available to your tenant.
- Trigger a workflow run via the API.
- Poll a run's status and retrieve its results.
