# Core Concepts

A short glossary of the terms you'll encounter while using the platform.

| Term         | Description                                                                                              |
| ------------ | -------------------------------------------------------------------------------------------------------- |
| **Tenant**   | An isolated workspace dedicated to a single organization. Your data is kept separate from other tenants. |
| **Agent**    | An AI-powered component that performs a task or responds to requests.                                    |
| **Workflow** | A defined sequence of steps that automates a business process.                                           |
| **Task**     | A unit of work executed as part of a workflow.                                                           |
| **Run**      | A single execution of a workflow, with its own status and history.                                       |

## How they fit together

A **workflow** is made up of one or more **tasks**, which may use **agents** to
do their work. Each time a workflow executes, it produces a **run** that you can
monitor and review.

Everything operates within your **tenant**, ensuring your organization's data
and activity remain isolated.
