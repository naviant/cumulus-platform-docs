# Release Notes

This page is the public changelog for the Cumulus Platform.

## Latest update

### 2026-06-18

- **Tenant lifecycle improvements:** Expanded tenant onboarding and offboarding
  automation, including safer networking teardown behavior, hard-delete
  workflow improvements, and stronger recovery for re-onboard scenarios.
- **Platform reliability:** Added broader live-tenant smoke coverage with
  timing visibility, more robust tenant selection, and execution timeout
  controls to improve operational confidence.
- **Core backend evolution:** Continued backend rebuild work to support the
  tenant-coded task framework.
- **Delivery efficiency:** Improved CI quality-job gating based on changed paths
  to speed up feedback and reduce unnecessary pipeline work.
- **Security and compliance:** We are preparing for our upcoming **SOC 2 Type
  1** security audit and continue aligning controls, evidence collection, and
  operational readiness accordingly.

## Previous updates

### 2026-01-01

- Naviant leadership and board elevated platform investment and committed to a

- This rebuild was prioritized to support the diverse workloads customers are

### 2025-04-01
